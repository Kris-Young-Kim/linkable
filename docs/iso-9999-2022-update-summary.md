### 1. ISO 매핑 테이블 업데이트 (`core/matching/iso-mapping.ts`)

**변경 전 → 변경 후:**

| 변경 전 코드 | 변경 후 코드 | 설명 |
|------------|------------|------|
| `18 12 10` | `18 30` | 수직 접근성 보조기기 (경사로, 승강기) |
| `15 09 13` | `15 09` | 식사 및 음주 보조기기 |
| `12 18 06` | `12 31` | 체위 변경 보조기기 |
| `12 12` | `12 06` / `12 03` / `12 08` | 보행 보조기기 (양팔/한팔/안내 지팡이) |
| `30 99 99` | `30 03` | 놀이 보조기기 |

**주요 개선사항:**
- ISO 9999:2022 표준에 맞는 정확한 Subclass 코드 사용
- ICF 코드 매핑 규칙 확장 (더 많은 조합 지원)
- 매핑 규칙 5개 → 15개로 확장

## ISO 9999:2022 주요 변경사항 (2016 → 2022)

1. **클래스 05 삭제**: 6판에서 7판으로 업데이트되면서 클래스 05가 삭제됨
2. **클래스 09, 12, 22 대폭 수정**: 구조 및 분류 체계 개선
3. **새로운 제목 추가**: 약 23개의 새로운 제목 추가
4. **제목 변경**: 116개의 제목이 변경됨 (소규모 편집 수정 포함)

## 코드 형식 규칙

ISO 9999:2022는 다음 형식을 사용합니다:

- **Subclass 레벨**: 4자리 숫자 (예: `1830`, `1509`, `1231`)
- **표시 형식**: 가독성을 위해 공백 포함 (예: `"18 30"`, `"15 09"`, `"12 31"`)
- **Division 레벨**: 6자리 숫자 (예: `183001`, `150901`, `123101`)

현재 프로젝트에서는 **Subclass 레벨**을 사용하며, 표시는 공백 포함 형식을 사용합니다.
