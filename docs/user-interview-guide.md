# 사용자 인터뷰 가이드 및 일정표

이 문서는 LinkAble의 사용자 인터뷰를 위한 가이드와 일정표를 포함합니다.

## 인터뷰 목적

LinkAble의 사용자 인터뷰는 다음을 목적으로 합니다:

1. **사용자 경험 파악**: 실제 사용자가 플랫폼을 어떻게 사용하는지 이해
2. **Pain Point 발견**: 사용 중 겪는 어려움과 불만 파악
3. **개선 사항 도출**: 사용자 피드백 기반 기능 개선 방향 수립
4. **사용자 여정 이해**: 상담 시작부터 추천 확인, 평가 제출까지의 전체 여정 파악

## 인터뷰 대상자

### 1. Primary Target (우선순위: 높음)

#### 1.1 고령자 당사자 (65세 이상)
- **인원**: 5-10명
- **특징**:
  - 실제 보조기기 사용자
  - 스마트폰 사용 경험 다양
  - 일상생활 불편함 경험
- **인터뷰 포커스**: 접근성, 사용 편의성, 추천 정확도

#### 1.2 보호자 (30-50대)
- **인원**: 5-10명
- **특징**:
  - 고령자 자녀 또는 보호자
  - 디지털 네이티브
  - 효도 소비 경험
- **인터뷰 포커스**: 신뢰도, 전문성, 만족도

### 2. Secondary Target (우선순위: 중간)

#### 2.1 복지용구 사업소 상담사
- **인원**: 3-5명
- **특징**:
  - 전문 상담 경험
  - 기존 상담 프로세스 이해
  - AI 도구 활용 의향
- **인터뷰 포커스**: 업무 효율성, 도구 유용성, 개선 사항

#### 2.2 요양보호사
- **인원**: 3-5명
- **특징**:
  - 현장 경험 풍부
  - 보조기기 사용자 관찰 경험
  - 실용적 관점
- **인터뷰 포커스**: 실용성, 현장 적용 가능성

## 인터뷰 일정표

### Phase 1: 준비 단계 (1주차)

#### 1.1 인터뷰 대상자 모집
- **기간**: 월요일 ~ 수요일
- **방법**:
  - 온라인 커뮤니티 공지 (뇌졸중 환우회, 파킨슨병 카페 등)
  - 복지용구 사업소 협조 요청
  - 기존 사용자 중 자원자 모집
- **목표**: 총 20-30명 모집

#### 1.2 인터뷰 가이드 최종화
- **기간**: 목요일 ~ 금요일
- **작업**:
  - 인터뷰 질문 리스트 확정
  - 인터뷰 시나리오 작성
  - 동의서 및 개인정보 처리 방침 준비

### Phase 2: 인터뷰 진행 (2-4주차)

#### 2.1 1차 인터뷰 (고령자 당사자)
- **기간**: 2주차 월요일 ~ 수요일
- **인원**: 5명
- **방식**: 온라인 화상 회의 또는 대면
- **소요 시간**: 1인당 60분
- **일정 예시**:
  - 월요일: 10:00, 14:00, 16:00
  - 화요일: 10:00, 14:00
  - 수요일: (예비 일정)

#### 2.2 2차 인터뷰 (보호자)
- **기간**: 2주차 목요일 ~ 금요일
- **인원**: 5명
- **방식**: 온라인 화상 회의
- **소요 시간**: 1인당 45분
- **일정 예시**:
  - 목요일: 19:00, 20:00, 21:00
  - 금요일: 19:00, 20:00

#### 2.3 3차 인터뷰 (복지용구 사업소 상담사)
- **기간**: 3주차 월요일 ~ 화요일
- **인원**: 3명
- **방식**: 대면 또는 온라인
- **소요 시간**: 1인당 90분
- **일정 예시**:
  - 월요일: 14:00, 16:00
  - 화요일: 14:00

#### 2.4 4차 인터뷰 (요양보호사)
- **기간**: 3주차 수요일 ~ 목요일
- **인원**: 3명
- **방식**: 온라인 화상 회의
- **소요 시간**: 1인당 60분
- **일정 예시**:
  - 수요일: 19:00, 20:00
  - 목요일: 19:00

#### 2.5 추가 인터뷰 (필요 시)
- **기간**: 4주차
- **인원**: 5-10명 (1-2차 인터뷰 보완)
- **목적**: 추가 질문 또는 심화 인터뷰

### Phase 3: 분석 및 정리 (5주차)

#### 3.1 인터뷰 내용 정리
- **기간**: 월요일 ~ 수요일
- **작업**:
  - 인터뷰 녹음/녹화 내용 전사
  - 주요 인사이트 추출
  - 공통 패턴 분석

#### 3.2 리포트 작성
- **기간**: 목요일 ~ 금요일
- **작업**:
  - 인터뷰 결과 리포트 작성
  - 개선 사항 도출
  - 우선순위 설정

## 인터뷰 가이드

### 인터뷰 전 준비

#### 1. 환경 설정
- [ ] 조용한 공간 확보
- [ ] 인터넷 연결 확인 (온라인 인터뷰 시)
- [ ] 녹음/녹화 장비 준비
- [ ] 인터뷰 가이드 및 질문 리스트 준비
- [ ] 동의서 및 개인정보 처리 방침 준비

#### 2. 인터뷰 대상자 준비
- [ ] 인터뷰 일정 확인 및 리마인더 발송
- [ ] 인터뷰 목적 및 절차 안내
- [ ] 동의서 사전 검토 요청

### 인터뷰 진행 절차

#### 1. 오프닝 (5분)
- 인터뷰 목적 설명
- 녹음/녹화 동의 확인
- 인터뷰 진행 방식 안내
- 질문이 있으면 언제든 말씀해달라고 안내

#### 2. 사용자 배경 파악 (10분)
- 연령, 거주지, 가족 구성
- 보조기기 사용 경험
- 스마트폰/인터넷 사용 능력
- LinkAble 사용 경험 (있는 경우)

#### 3. 사용 경험 탐색 (30-40분)
- LinkAble 첫 사용 경험
- 상담 진행 과정
- 추천 결과 확인 경험
- K-IPPA 평가 제출 경험 (있는 경우)
- 어려웠던 점 및 불만사항
- 만족했던 점

#### 4. 개선 사항 논의 (10-15분)
- 추가로 원하는 기능
- 개선이 필요한 부분
- 다른 서비스와의 비교
- 추천 의향

#### 5. 클로징 (5분)
- 추가 질문 확인
- 감사 인사
- 향후 연락 가능 여부 확인

### 인터뷰 질문 리스트

#### A. 사용자 배경 질문

1. **기본 정보**
   - 연령대는 어떻게 되시나요?
   - 거주 지역은 어디신가요?
   - 가족 구성은 어떻게 되시나요?

2. **보조기기 사용 경험**
   - 보조기기를 사용해보신 경험이 있으신가요?
   - 어떤 보조기기를 사용하고 계신가요?
   - 보조기기를 어떻게 구매하셨나요? (온라인/오프라인)

3. **디지털 사용 능력**
   - 스마트폰을 자주 사용하시나요?
   - 카카오톡, 인터넷 검색 등을 사용하시나요?
   - 온라인 쇼핑 경험이 있으신가요?

#### B. LinkAble 사용 경험 질문

1. **첫 사용 경험**
   - LinkAble을 어떻게 알게 되셨나요?
   - 첫 사용 시 어떤 생각이 드셨나요?
   - 사용하기 쉬웠나요? 어려웠나요?

2. **상담 진행**
   - AI 상담을 진행하면서 어떤 느낌이었나요?
   - 질문이 이해하기 쉬웠나요?
   - 답변에 만족하셨나요?

3. **추천 결과**
   - 추천된 보조기기를 확인해보셨나요?
   - 추천 결과가 자신의 상황과 맞다고 생각하시나요?
   - 추천 이유가 이해되셨나요?

4. **K-IPPA 평가**
   - K-IPPA 평가를 제출해보셨나요?
   - 평가 과정이 어려우셨나요?
   - 평가 결과가 도움이 되셨나요?

#### C. 개선 사항 질문

1. **불만사항**
   - 사용하면서 불편했던 점이 있나요?
   - 개선이 필요한 부분이 있다면 무엇인가요?
   - 에러나 문제가 발생한 적이 있나요?

2. **추가 기능**
   - 추가로 원하는 기능이 있나요?
   - 다른 서비스에서 본 좋은 기능이 있나요?
   - 전문가 상담이 필요하다고 생각하시나요?

3. **비교**
   - 다른 보조기기 추천 서비스를 사용해보셨나요?
   - LinkAble과 비교했을 때 어떤 점이 좋았나요?
   - 어떤 점이 부족하다고 생각하시나요?

#### D. 추천 의향 질문

1. **추천 의향**
   - 주변 지인에게 LinkAble을 추천하시겠나요?
   - 추천하신다면 이유는 무엇인가요?
   - 추천하지 않으신다면 이유는 무엇인가요?

2. **지불 의향**
   - LinkAble이 유료 서비스라면 사용하시겠나요?
   - 적정 가격대는 어느 정도라고 생각하시나요?
   - 어떤 기능에 대해 지불하시겠나요?
